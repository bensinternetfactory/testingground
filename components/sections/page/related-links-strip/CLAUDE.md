# related-links-strip

Shared related-links strip for equipment financing pages.

## Usage

```tsx
import { RelatedLinksStrip } from "@/components/sections/page/related-links-strip";

<RelatedLinksStrip config={config.relatedLinks} />
```

## Props

| Prop | Type | Description |
|---|---|---|
| `config` | `RelatedLinksStripConfig` | Link list rendered in the strip |

## Server/Client Boundary

- `RelatedLinksStrip.tsx` — server component
